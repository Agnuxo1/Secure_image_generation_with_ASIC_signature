Secure Image Generation with ASIC Signature — Release v1.0.0

This release includes:

- Source code for the signature engine, RS encoder, LSB embedder and verifier.
- Paper PDF and HTML: "Robust ASIC-Based Image Authentication Using Reed–Solomon LSB Watermarking".
- Dataset: `Originals/` and `proofs/` example images and results used in experiments.
- Benchmark runner and CI workflow to execute and upload results to Weights & Biases (W&B).
- LICENSE (MIT) and dataset metadata (dataset/README.md).
